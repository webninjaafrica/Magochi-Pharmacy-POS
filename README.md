## 📖 Quick User Guide

**Point of Sale:**
- Double-click any product to add to cart
- Enter quantity when prompted
- Click Print Receipt, Save as PDF, or HTML Invoice

**Stock Management:**
- Add Product: Fill details → Click "Add Product"
- Edit: Select product → Modify → "Update Selected"
- Delete: Select → "Delete Selected" → Confirm

**Branch Management:**
- Add Branch: Name, location, phone → "Add Branch"
- Switch Branch: Use dropdown at top-right
- Delete: Select → "Delete Selected"

**Printer Setup:**
- Go to Printer Settings tab
- Connection types: windows, cups, network, serial
- Select printer → "Set as Default"
- Click "Test Print" to verify

**Receipts:**
- Go to Receipts tab
- Search by receipt number or date range
- Select receipt → "Reprint Receipt" or "Generate PDF"

**Reports:**
- Sales Report: Select date range → Generate
- Stock Report: Click "Generate Stock Report"
- Audit Log: Menu → Reports → Audit Log

**Backup & Restore:**
- Manual Backup: File → Create Local Backup
- Auto Backup: Daily automatic backup to backups folder
- Restore: Backup Manager → Double-click backup file
- Cloud Backup: Premium only

**Customization:**
- Company Logo: File → Company Settings → Upload Logo
- Theme Color: File → Company Settings → Choose Color
- Receipt Settings: File → Receipt Settings → Customize header/footer/tax rate

## 🔧 Troubleshooting

**Application won't start:** Run as Administrator; check antivirus quarantine

**Printer not printing:** Verify printer is default in Windows; test with Notepad

**PDF generation fails:** Install ReportLab (included in installer)

**Login failed:** Use correct credentials; case-sensitive

**Cloud sync offline:** Premium feature; check internet connection
